# qt-device-manager
A Qt Widgets based device management system developed for learning Qt/C++ desktop application development. The project covers multi-window navigation, signal-slot communication, dynamic table operations, object lifecycle management, and basic CRUD functionality.
This project developed an intelligent video surveillance equipment management system based on Qt 6 and C++. The system implements user login authentication, equipment information management, JSON data persistence, video source binding, and real-time video preview functions. It employs a signal-slot mechanism for multi-window communication, uses QMediaPlayer for local video and network streaming media playback, and supports equipment status monitoring and extended video screenshot functionality.
